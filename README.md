# NumPy AI Foundations 🚀

### Overview
This repository contains essential **NumPy** exercises designed for high-performance computing in **Artificial Intelligence**. 

### Key Features:
- **Matrix Calculus:** Implementing Dot Products and Transpositions.
- **Boolean Masking:** Filtering datasets without Python loops (Vectorization).
- **Statistical Generation:** Creating Normally Distributed arrays for Weight Initialization.

### Performance Environment:
Tests were executed on an **ASUS TUF Gaming F16** (Intel Core i7-14650HX), leveraging NumPy's integration with **Intel MKL** for optimized floating-point operations.
